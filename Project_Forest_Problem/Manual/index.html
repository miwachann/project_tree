ソフトウェア工学IIの小テスト「樹状整列」マニュアル

インストール（実行ファイルの作り方）

  1. tree.zipのアーカイブファイルを適当な場所に展開する。
    2. $ unzip tree.zip
  2. 展開して得られるtreeというディレクトリへ移る。
    3. $ cd tree
  3. 中身を確認する。
    4. $ ls *
  4. makeする。
    5. $ make
  5. tree.jarが生成されていることを確かめる。
    6. $ ls *
  6. 生成したtree.jarをアプリケーション（tree.app）の中にインストールする。
    7. $ make install

実行方法（実行ファイルの動かし方）

実行するためのコマンドは以下の通りである。

java -jar tree.jar

Mac OS Xの場合には、アプリケーションMVC.appをダブルクリックするほうが簡便である。

tree.app

また、openコマンドでMVC.appを開いても実行可能である。

open tree.app
